# College Attendance Management System

A web-based attendance management system for colleges with teacher and student dashboards.

## Features

### 👨‍🏫 Teacher Dashboard

- Add new students with name, ID, email, and class
- Mark attendance for different subjects
- View class statistics and overall attendance
- Manage student records

### 👨‍🎓 Student Dashboard

- View personal attendance information
- Check subject-wise attendance breakdown
- See recent attendance records
- Track attendance percentages

### 🔐 Authentication System

- Role-based login (Teacher/Student)
- Secure access to respective dashboards

## Getting Started

### Default Credentials

**Teacher Access:**

- Username: `teacher`
- Password: `teacher123`

**Student Access:**

- Username: Your Student ID
- Password: `student123`

### How to Use

1. **Login**: Open `index.html` and select your role
2. **Teacher**: Add students and mark attendance
3. **Student**: Login with your ID to view attendance

## File Structure

- `index.html` - Login page
- `teacher.html` - Teacher dashboard
- `student.html` - Student dashboard
- `display.html` - Display dashboard
- `script.js` - Main JavaScript functionality
- `styles.css` - CSS styling
